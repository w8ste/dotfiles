### Installation
These are the pieces of software i use:
- alacritty
- btop
- dunst
- flameshot
- htop
- i3
- kitty
- lazygit
- neofetch
- nvim
- picom
- polybar
- rofi
- tmux
- betterlockscreen
- starship

You will also have to install a [Nerd Font](https://www.nerdfonts.com/)
